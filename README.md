# WEBSITE-PENGENALAN-CERITA-RAKYAT-INTERAKTIF
website ini melestarikan dan mengenalkan cerita rakyat interaktif yang menarik 
langkah langkah untuk menyiapkan program adalah sebagai berikut:
1. Download file "index.html"
2. Buka link https://drive.google.com/drive/folders/1LcRZtSMjxcb344GtNjpq7D47_9fLWL2w?usp=sharing download file "images" dan "audio"
3. Buka file yang telah di download
4. Pindahkan file .jpg yang telah di download dalam satu file beri nama "images"
5. Lakukan yang sama untuk file audio .mp3 namun untuk file nya bernama "audio"
6. Buatlah file baru (bebasa untuk nama) dan jadikan file "images" & "audio" yang berisi gambar dan "index" dalam satu folder
7. Setelah mendownload semua file coba untuk memasukan program dari file "index.html" kedalam VSC (VISUAL STUDIO CODE)
8. lalu coba untuk menjalankan program nikmati program demo sederhana inii! jangan lupa untuk berikan saran dan masukan untuk demo sederhana kami terimakasihhh!!!
