# WEBSITE-PENGENALAN-CERITA-RAKYAT-INTERAKTIF
website ini melestarikan dan mengenalkan cerita rakyat interaktif yang menarik 
langkah langkah untuk menyiapkan program adalah sebagai berikut:
1. Download file "index.html"
2. Download file "images.zip"
3. Buka file Zip yang telah di download
4. Copy file .png yang ada dalam file zip
5. Buatlah File baru bernama "images" dan tempel foto di dalam file tersebut
6. Buatlah file baru dan jadikan file "images" yang berisi gambar dan "index" dalam satu folder
7. Setelah mendownload kedua file coba untuk memasukan program dari file "index.html" kedalam VSC (VISUAL STUDIO CODE)
8. lalu coba untuk menjalankan program nikmati program demo sederhana inii! jangan lupa untuk berikan saran dan masukan untuk demo sederhana kami terimakasihhh!!!
